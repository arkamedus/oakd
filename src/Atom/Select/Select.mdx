# Select

The `Select` component handles both simple pickers and categorized menus. It
supports controlled and uncontrolled usage, keyboard selection, custom option
content, and visual sizing that matches the rest of the oakd input family.

<Primary />
<Controls />

## What To Validate

- `onChange` reports the selected value rather than raw DOM events.
- Controlled `value` flows keep the trigger label in sync with application state.
- `categoryOrder` groups options in a predictable order when menus get larger.
