# Collapsible

The `Collapsible` component reveals supporting content without forcing the page
to navigate away or grow permanently. It fits release notes, secondary settings,
knowledge-base sections, and anywhere a team needs progressive disclosure.

<Controls />

## What To Validate

- The trigger exposes `aria-expanded` so state changes are clear.
- The trigger is the only interactive control in the header row.
- `defaultOpen` matches the initial visual state.
- Inner spacing should be provided intentionally, typically with `Content pad`.
