# MarkdownEditor

The `MarkdownEditor` component pairs the oakd `CodeArea` editor with the oakd
`MarkdownRenderer` preview so edit and preview modes stay visually consistent.

The toolbar uses oakd icons, and preview mode renders through the same
`MarkdownRenderer` component used elsewhere in the library. That keeps custom
icon syntax, `THINK` segments, and streaming states consistent between authored
and rendered markdown.
