# SpeechToText

The `SpeechToText` component provides a compact recording trigger for converting
spoken language into text using the Web Speech API. The component itself stays
focused on recording state; rendering the captured transcript can be handled by
the parent when needed.

## Props

- **buttonText** (string): The text displayed on the button. Defaults to "Start Speaking".
- **listeningText** (string): The text displayed while recording is active.
- **onChange** (function): Callback function that is called with the transcribed text when it changes.
- **disabled** (boolean): Disables the trigger button.
- **size** (string): Matches the button sizing scale.
