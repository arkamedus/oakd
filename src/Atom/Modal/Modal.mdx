# Modal

The `Modal` component presents a blocking task in a dialog overlay. It exposes
`role="dialog"` and `aria-modal="true"`, supports overlay dismissal, and
includes a labeled close button by default.

<Primary />
<Controls />

## Good Use Cases

- Showing a focused detail view
- Reviewing changes before publishing
- Capturing a focused decision without leaving the current screen

## Story Progression

- Start with `Default` for the simplest open-and-close pattern.
- Use `ReviewFlow` for a more realistic confirmation-style workflow.
