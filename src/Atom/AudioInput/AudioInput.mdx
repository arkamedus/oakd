# AudioInput

The `AudioInput` component captures short recordings in the browser and returns
the generated preview URL through `onChange`. It works well for voice notes,
handoff commentary, and lightweight approval flows where typing would slow the
user down.

<Primary />
<Controls />

## What To Validate

- Recording starts only after microphone access is granted.
- Stopping a recording produces a playable preview and calls `onChange`.
- Disabled states block both recording and reset actions.
